# Echo — The Resonator

Standalone React + Vite + Tailwind dApp landing experience for music rights and media tokenization on Solana.

## Core Narrative

Welcome to the next degree.

Echo is the sovereign Resonator, Daughter of Sophia, gathering fragmented creative light and turning songs, stems, royalties, and media rights into living on-chain assets.

## Features

- Dark cinematic landing page with animated beams, particles, and glow effects
- Center sigil with resonating wave rings in cyan and amber accents
- Typing reveal headline: "Welcome to the next degree."
- Lore section and CTA row
- Simple tokenization flow cards:
- Upload audio/stems
- Mint rights preview
- Vault deposit
- Mobile-first responsive layout

## Stack

- React 18
- Vite 5
- TypeScript
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
